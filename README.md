# Azure Blob Storage Example

This example project uses Azurite, Node, Express.js, React, Vite for a full-stack demonstration of using Azure Blob Storage.

## Running Locally

### Run Azurite

```bash
docker compose up -d
```

### Run Backend

```bash
cd backend
pnpm install
pnpm run dev
```

### Run Frontend

```bash
cd frontend
pnpm install
pnpm run dev
```

Open the frontend app in browser http://localhost:3000.
